# DevTrack

A modern Android application built with Kotlin and Jetpack Compose that helps students and developers organize projects, track learning progress, and visualize productivity through analytics.

---

## Features

- 🔐 Secure Email Authentication
- 📊 Progress Dashboard
- 📁 Project Management
- 📝 Learning Journal
- 📈 Productivity Analytics
- ☁️ Cloud Synchronization
- 🤖 AI-Powered Insights (Planned)

---

## Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- MVVM Architecture
- Navigation Compose
